<h1> Лабораторная работа №2.3 "Очередь задач: итераторы и генераторы" </h1>
<hr>
<p>Программа содержит итератор для работы с задачами из файла</p>

<p>Поддерживает следующие команды:</p>
<ul>
    <li> python -m src.main show-tasks [filename] - выводит задачи из файла </li>
    <li> python -m src.main status-filter [filename] [status] - выводит задачи из файла с определённым статусом </li>
    <li> python -m src.main priority-filter [filename] [priority] - выводит задачи из файла с определённым приоритетом </li>
    <li> pytest --cov=. --cov-report=term-missing - запуск тестов </li>
</ul>

<h2> Итераторы </h2>

<h3>TaskQueue</h3>
    <p>При инициализации принимает имя файла, в котором содержатся задачи. Во время итерирования вызывается метод __next__(), который открывает файл (если он был закрыт), считывает строку из него, парсит её и возвращает в виде экземпляра Task. Когда встречается пустая строка или конец, файл закрывается и поднимается StopIteration. Один и тот же итератор можно вызывать повторно. Также есть методы status_filter и priority_filter, которые возвращают фильтрующий итератор FilteredTaskIterator.</p>

<h3>FilteredTaskIterator</h3>
    <p>При инициализации принимает имя файла, в котором содержатся задачи, имя поля и его, по которым фильтруются задачи. Во время итерирования вызывается метод __next__(), который открывает файл (если он был закрыт), считывает строки из него и парсит их пока не найдёт нужную или не дойдёт до конца, и возвращает в виде экземпляра Task. Когда встречается пустая строка или конец, файл закрывается и поднимается StopIteration. Один и тот же итератор можно вызывать повторно.</p>
